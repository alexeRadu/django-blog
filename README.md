# Create the Project
## In windows
```
$ cd <path>/djago-blog
$ python -m venv venv/
$ venv/Scripts/activate
$ pip install django
$ pip install django-crispy-forms
$ pip install Pillow
```

## In Linux (Ubuntu)
```
$ cd <path>/djago-blog
$ python -m venv venv/
$ source venv/Scripts/activate
$ pip install django
$ pip install django-crispy-forms
$ pip install Pillow
```
